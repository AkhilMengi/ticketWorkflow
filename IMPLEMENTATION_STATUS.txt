🎯 HOW IT WORKS
═══════════════════════════════════════════════════════════════

1️⃣  LOAD GENERIC SUGGESTIONS
    From: suggestions.txt
    Examples:
    - "Check customer details"
    - "Rebill the account"
    - "Close the case"
    - "Escalate to team"
    - "Document findings"

2️⃣  AI ANALYZES ISSUE + SUGGESTIONS
    Input: Customer issue descriptions
    Process: LLM reads issue + suggestions + mapping rules
    Output: Intelligent action mapping

3️⃣  MAP SUGGESTIONS TO SYSTEM ACTIONS
    Examples:
    "Check details" → create_case (investigate issue)
    "Rebill account" → apply_billing_adjustment (credit/refund)
    "Escalate team" → escalate_to_team (human review)

4️⃣  EXECUTE MULTIPLE ACTIONS
    Primary action (high confidence)
    Secondary action (supporting)
    Tertiary action (additional)

5️⃣  RETURN TRANSPARENT RESULTS
    Shows which suggestion triggered each action
    Includes confidence scores
    Audit trail for tracking
